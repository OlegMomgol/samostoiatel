# samostoiatel
Программа для учета личных финансов, позволяющее пользователям вести учет доходов и расходов. 
Основные возможности включают добавление операций с указанием типа (доход/расход), категории, суммы, даты и описания; удаление операций по ID; расчет финансовой статистики (общие суммы, баланс, распределение по категориям);
Приложение имеет понятный интерфейс с меню, обрабатывает ошибки ввода и автоматически загружает сохраненные данные при запуске. 
Данные хранятся в виде с использованием вектора, что обеспечивает удобное управление записями.
